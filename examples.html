-097\xa0\xa0\xa0THE USE OF LONG VARCHAR OR LONG VARGRAPHIC IS NOT ALLOWED\nIN THIS CONTEXT

Explanation

The statement attempted to use the LONG VARCHAR\nor LONG VARGRAPHIC syntax. This syntax cannot be used for the following\nstatements:
  • CAST specification syntax
    • \n
  • CREATE DISTINCT TYPE
    • \n
  • CREATE FUNCTION
    • \n
  • CREATE PROCEDURE
    • \n
  • ALTER FUNCTION
    • \n
  • COMMENT ON FUNCTION
    • \n
  • GRANT EXECUTE ON FUNCTION
    • \n
  • REVOKE EXECUTE ON FUNCTION
    • \n
  • DROP
    • \n
\n
\n

Use the VARCHAR or VARGRAPHIC syntax specifying an explicit\nlength as required.

\n
\n

System action

The statement cannot be processed.

\n
\n

Programmer response

Correct the statement.

\n
\n

SQLSTATE

42601
